secureshout by krig

The project goal is to enable anonymous communication without the exit node problem.

The main idea is to allow truly anonymous communication. One idea is to pervert public/private keysigning by signing each outgoing message piecemeal with N randomly selected public keys (retrieved from peers), so that the message can only be reassembled using all those separate keys. Pieces are circulated between peers in both encrypted and decrypted states, and messages are reassembled and published when possible. This way, no one can know where the message originated. More concepts to be added (and more problems need solving).
